webGraphEd help

Actions on the canvas

  • Double click: create a node.
  • Drag: pan (move the layout).
  • Mouse wheel: zoom in and out.

Actions on a node

  • Click: select/unselect node).
  • Double click: change properties.
  • Drag: drag a node.
  • Ctrl + Drag: link a node to another (not allowed when there are some collapsed nodes).
  • Shift + click: collapse/uncollapse a node (only allowed in trees).
  • [DEL], [BACKSPACE]: delete node if selected.

Actions on a label

  • Double click: change label name.
