webGraphEd help

Actions on the canvas

  • Double click: create a node.
  • Drag: pan (move the layout).
  • Mouse wheel: zoom in and out.

Actions on a node

  • Double click: change properties (not implemented).
  • Drag: drag a node.
  • Ctrl + Drag: link a node to another.

Actions on a label

  • Double click: change label name.
