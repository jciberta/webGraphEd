webGraphEd
==========

webGraphEd is an open source software for graph visualization and manipulation.


Deployment
----------

If you want to use the application, just get the .min files:
- webGraphEd.min.html
- webGraphEd.min.js
- webGraphEd.min.css

Development
-----------

To reduce the application: 

    compact.sh

To generate the documentation: 

    gendoc.sh
